# 🤖 Simple Python ChatBot

A beginner-friendly chatbot built using Python.  
This chatbot responds to greetings, basic conversations, and common user inputs using conditional statements.

## Features

- Greets users
- Responds to:
  - Hello / Hi
  - Good morning / evening / night
  - "How are you?"
  - Positive and negative feelings
  - Thank you messages
- Handles unknown inputs
- Exit using:
  - bye
  - goodbye
  - quit
- Clean chat-style interface
